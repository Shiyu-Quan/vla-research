# VLA Acceleration Taxonomy

Maintain technique families that recur across multiple papers.

Suggested starting categories:

- Model and token compression
- Action representation and chunking
- Diffusion or flow action generation
- Parallel or speculative action decoding
- Conditional compute and layer skipping
- Temporal caching and redundancy
- Runtime pipelining and multi-rate scheduling
- Dedicated accelerator and memory-system design

Add a category only when it helps compare more than one work.
